Keyboard Shortcuts

General

  1. Select a Problem: Ctrl + O
  2. Next Problem: Ctrl + N
  3. Check Answer: Ctrl + K
  4. Save Answer: Ctrl + S
  5. Print Answer: Ctrl + P
  6. Close Window: Alt + F4

Derivations

  1. Show/Unshow: Ctrl + Shift + S
  2. Add New Line: Ctrl + Shift + M
  3. Delete Entire Line: Alt (Fn + Option) + Del
  4. Therefore: Ctrl + Shift + T

Chapter 1

  1. Conditional Arrow: Ctrl + Shift + C
  2. Negation Symbol: Ctrl + Shift + N

Chapter 2

  1. Conjunction: Ctrl + Shift + A
  2. Disjunction: Ctrl + Shift + O
  3. Biconditional: Ctrl + Shift + B

Chapter 3

  1. Universal Quantifier: Ctrl + Shift + U
  2. Existential Quantifier: Ctrl + Shift + E
